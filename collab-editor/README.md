# Real-Time Collaborative Editor

A Google Docs-like collaborative text editor built using WebSockets for real-time updates.

## Features

- Real-time collaboration
- Multi-user support
- Built with React, Node.js, and Socket.io

## How to Run

### Backend

```bash
cd server
npm install
node index.js
```

### Frontend

```bash
cd client
npm install
npm start
```
